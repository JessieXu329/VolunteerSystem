# VolunteerSystem
A volunteer event registration platform built with Django and Vue. Users can browse, filter, and search for activities. Admins can create events and view registration details.  本项目是一个使用 Django 和 Vue 开发的志愿活动报名平台。用户可以浏览、筛选、搜索活动，管理员可以发布活动并查看报名详情。

## Features | 功能亮点
1.Support for searching and filtering events by category  
支持按分类搜索和筛选活动  
2.User-friendly event registration interface  
提供友好的活动报名页面  
3.Admin panel for creating events and managing participant information  
管理员后台可以发布活动、查看报名情况  
4.Real-time status updates for events  
活动状态自动更新（如报名截止、已满等）
